# IBE260-Oblig-2-SSB

By Kristoffer and Stine

This project is a simple Node.js Express application for the IBE260 course assignment 2. It includes server-side programming using JavaScript, TypeScript, Express, and JSONstat toolkit.

### Features
- Express Server Setup: Utilizes Express.js for handling HTTP requests and serving static files.
- Dynamic Checkbox Creation: Dynamically creates checkboxes based on predefined data.
- API Endpoint for SSB Data: Implements a POST endpoint to fetch data from the Statistics Norway API.
- Table Creation and Statistics: Generates tables from data and calculates statistics such as average, min, max, and median.

### Pre-req
[Node.js](https://nodejs.org/en/download) and [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) (included with Node.js)

### How to use

1. Clone this repository to your local machine `git clone https://github.com/krihop/IBE260-Oblig-2-SSB.git`

2. Install dependencies: `npm install`

3. Start the server `npm run start`

4. Server runs on port [3000](http://localhost:3000)
